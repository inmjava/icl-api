# Projeto

## Introdução

Este README/LEIAME é o ponto de partida em termos da documentação do projeto. A documentação do projeto deve ser armazenada nos diretórios de nomes "docs ...", sendo que cada um destes contém um README.md que explica sua estrutura e propósito.

Este template contém o arquivo .gitignore para ignorar arquivos comuns como .env, .classpath etc.

Mais informações podem ser encontradas em: "Corporativo->Tecnologia da Informação->Processos->Gerência de Configuração de Software->Regras de Gerência de Configuração para GIT"
